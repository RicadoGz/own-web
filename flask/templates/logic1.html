I developed a comprehensive web application for stock analysis using the Relative Strength Index (RSI), integrating various technologies and skills to create an efficient and innovative platform. Compared to traditional methods of searching for RSI, my application reduces search time by over 70%. Additionally, by leveraging asynchronous programming, I achieved a further 30% reduction in update times, ensuring a highly responsive and efficient user experience. I utilized HTML/CSS/JavaScript to create responsive and interactive user interfaces. The backend was developed using Python/Django for handling requests, processing data, and rendering templates. Although not explicitly mentioned in the provided files, I integrated SQL for potential database interactions. I managed version control using Git/GitHub, ensuring efficient collaboration and code management. To handle multiple tasks concurrently and improve performance for large data sets, I utilized Asynchronous Coroutines (Asyncio), as mentioned in stock.html. I used Web Scraping techniques in data.py to fetch financial data from the web. I employed HTTP protocols for handling web requests and responses, and utilized Data Structures in Python scripts to efficiently manage and process stock data.
